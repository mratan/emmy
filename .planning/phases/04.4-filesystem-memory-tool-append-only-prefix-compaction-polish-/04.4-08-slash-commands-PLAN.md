          const trimmed = args.trim();

        },
      });

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator → /compact args | User text appended to customInstructions; pi's compact() runs the round-trip |
| operator → /clear confirm | Destructive action; confirm dialog protects against accidental keystroke |
| handler → emitEvent | In-process JSONL/OTLP best-effort; never blocks command on telemetry hiccup |
| /clear → newSession | New session boots a fresh prompt assembly per D-3X — invariant preserved across boundary |
