    Implementation outline (LITERAL steps; do not abridge):

    1. **Normalize input first.** Reject if `logicalPath` contains:
       - ` ` (null byte) → return `MemoryError("memory.traversal_blocked", "null byte injection")`
       - `\\` (backslash) → return `MemoryError("memory.traversal_blocked", "backslash not permitted")`
       - URL-encoded dots `%2e%2e` (case-insensitive) or slash `%2f` → return `MemoryError("memory.traversal_blocked", "URL-encoded path component")`
       - Literal substring `..` (any position) → return `MemoryError("memory.traversal_blocked", "parent-traversal segment")`
       - Empty string → return `MemoryError("memory.traversal_blocked", "empty path")`
       - Does not start with `/memories` → return `MemoryError("memory.traversal_blocked", "must start with /memories")`

    2. **Strip `/memories` prefix.** Remaining tail dispatches:
       - `""` (path was exactly `/memories`) → return `{ok:true, scope:"virtual", absPath:"", logicalPath}` (caller view-command handles top-level listing)
       - `/project` or `/project/...` → scope="project", inner = part after `/project`
       - `/global` or `/global/...` → scope="global", inner = part after `/global`
       - else → return `MemoryError("memory.traversal_blocked", "scope must be project or global")`

    3. **Resolve scope root.** For scope="project": rootRel = config.project_root, base = cwd. For scope="global": rootRel = config.global_root, base = HOME (homedir()).
       - If rootRel is null → return `MemoryError("memory.disabled", "scope <name> disabled in profile")` with code `"memory.disabled"`.
       - Expand `~/...` prefix in rootRel by replacing the leading `~` with `homedir()`.
       - rootAbs = `path.resolve(base, rootRel)` (absolute)

    4. **Compose candidate path.** `candidate = path.resolve(rootAbs, "." + inner)` — the leading `.` keeps `inner=""` resolving to rootAbs itself.

    5. **Containment check (pre-symlink).** Compute `normalize(candidate)` and assert it starts with `normalize(rootAbs) + sep` OR equals `normalize(rootAbs)`. If not, return traversal_blocked.

    6. **Containment check (post-symlink).** If `existsSync(candidate)`, compute `realpathSync(candidate)` and re-assert containment under `realpathSync(rootAbs)` (which we cache once per resolveMemoryPath call). Fall through if file doesn't exist (create path).

    7. **Blocked extension check.** Compute lowercase suffix of basename; if `config.blocked_extensions` (lowercased set) contains it, return `MemoryError("memory.blocked_extension", "extension <ext> is in blocked_extensions")`.

    8. Return `{ok:true, scope, absPath: candidate, logicalPath}`.
