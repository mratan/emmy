# Plan 04.6-02 — Pre-send scrubber library

**Wave:** 1 — no dependencies
**Layer:** Python (`emmy_serve/tools/scrubber.py`)
**Goal:** A small, fast, deterministic library that takes a prompt string and returns whether it matches any operator-configured "do not send" pattern (regex + glob-ish file-path detection). Reject-don't-redact policy (D-06) — caller (sidecar) returns an error to the model on match. The scrubber itself does NO IO; pattern config is loaded once at module import from a YAML file at a known location.

### Task 2 — GREEN: implementation

`emmy_serve/tools/scrubber.py`:

```python
"""Pre-send scrubber for ask_claude (Plan 04.6-02 / D-06).

Reject-don't-redact: caller gets a structured error on match, not a
silently-modified prompt. Pattern config is loaded once at import; reload
requires sidecar restart (intentional — config drift would be a bug source).

False positives are tolerable; false negatives are NOT (T-01 data exfil).
"""
from __future__ import annotations
from dataclasses import dataclass
from pathlib import Path
import re
import yaml

@dataclass(frozen=True)
class ScrubResult:
    clean: bool
    matched_class: str | None = None
    matched_excerpt: str | None = None  # short snippet of what triggered, NOT full content

# Compiled patterns; class -> compiled regex
_PATTERNS: dict[str, re.Pattern] = {}

def _load_config() -> None:
    cfg_path = Path(__file__).parent / "scrubber_config.yaml"
    raw = yaml.safe_load(cfg_path.read_text())
    for cls_name, pattern_str in raw["patterns"].items():
        _PATTERNS[cls_name] = re.compile(pattern_str)

_load_config()

def scrub(prompt: str) -> ScrubResult:
    """Return ScrubResult; .clean=False on first match. Order is
    deterministic (config-yaml ordering)."""
    for cls_name, pat in _PATTERNS.items():
        m = pat.search(prompt)
        if m:
            excerpt = m.group(0)
            # Excerpt is the matched substring, capped — for audit/error messages
            # without leaking full prompt context.
            return ScrubResult(
                clean=False,
                matched_class=cls_name,
                matched_excerpt=excerpt[:40] + ("…" if len(excerpt) > 40 else ""),
            )
    return ScrubResult(clean=True)
```

`emmy_serve/tools/scrubber_config.yaml`:

```yaml
# Default blocklist for ask_claude pre-send scrubber.
# Patterns are evaluated in order; first match wins.
# Operator can override by setting EMMY_SCRUBBER_CONFIG to an alternate path
# (NOT shipped in v1 — v1 uses this committed default).
patterns:
  aws_access_key_id: '\bAKIA[0-9A-Z]{16}\b'
  sk_prefixed_key: '\b(sk-(?:ant-)?[a-zA-Z0-9_\-]{20,})\b'
  slack_token: '\bxox[abprs]-[a-zA-Z0-9-]{10,}\b'
  github_token: '\bgh[pousr]_[a-zA-Z0-9]{36,}\b'
  jwt_or_bearer: '\beyJ[a-zA-Z0-9_-]{10,}\.eyJ[a-zA-Z0-9_-]{5,}\.[a-zA-Z0-9_-]{5,}\b'
  google_api_key: '\bAIza[0-9A-Za-z_\-]{35}\b'
  private_key_header: '-----BEGIN [A-Z ]*PRIVATE KEY-----'
  secret_path_env: '\b\.env(\.[a-z]+)?\b'
  secret_path_ssh: '~?/?\.ssh/(id_(rsa|ed25519|ecdsa|dsa)|authorized_keys|known_hosts)'
  secret_path_dir: '\b(secrets|credentials)/[a-zA-Z0-9._/-]+'
  secret_path_keyfile: '[a-zA-Z0-9_-]+\.(pem|key|p12|pfx|jks)\b'
```
