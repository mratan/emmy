## Decisions Made

- **Place runbook section between SearxNG and Langfuse, not under Phase 04.2 sidecar.** Operator-mental-model rationale: a reader asking "what egresses can Emmy do?" expects to find both adjacent. The Phase 04.2 sidecar section is about lifecycle (start / stop / status / profile), not egress. ask_claude is conceptually a cousin of web_search even though it lives on the same sidecar FastAPI process.
- **Default-OFF three-layer posture stated TWICE** (once in the runbook section header, once as a parenthetical in the CLAUDE.md thesis). The redundancy is deliberate; egress docs that mention default-OFF only once tend to be misread. Three independent gates (env / schema / enabled-flag) make the misreading "this thing is on by default" structurally impossible — operator must pass all three.
- **Cross-reference 04.6-07-OPERATOR-PROTOCOLS.md from the runbook**, not duplicate. OPERATOR-PROTOCOLS is the attestation record (replace `[ ]` with `[x] <date>` per operator gesture); duplicating its 7-SC walkthroughs in the runbook would create two-source-of-truth drift. Runbook gets the quick-reference 3-step pre-flight; OPERATOR-PROTOCOLS gets the full SC battery + sign-off.
- **Air-gap CI invisibility paragraph explicit about cloud upstream.** The Pitfall #5 extension calls out: "Air-gap CI invisibility is the same shape — STRICT validator gates outbound sockets from the *harness*; `claude` subprocess spawned from the sidecar is outside its perimeter." The reader should NOT come away thinking ask_claude is also loopback — it's not. The honesty hook is essential to the threat model.
- **ToS-policy compliance leg surfaced as a Pitfall #5 extension, not a separate Pitfall #21.** The existing Pitfall #5 "Hidden cloud dependencies" is the natural home — ask_claude is the answer to "why isn't this a hidden dep". Adding a 21st pitfall would dilute the 8-critical / 12-secondary structure documented in `.planning/research/PITFALLS.md`.
- **Reason-code → operator-action table over prose troubleshooting.** Direct lookup from observed symptom to fix. 7 rows = 7 reason codes from controller.py exhaustively. Future egress endpoints (e.g. `/ask-codex` if multi-provider escalation lands) should publish the same shape table — pattern established.
- **README Quickstart compactness over completeness.** The /ask-claude Quickstart entry is one line with the operator-friendly caveat; detailed walkthroughs live in the runbook. Reader's path: see /ask-claude in Quickstart → curious → click runbook link.
