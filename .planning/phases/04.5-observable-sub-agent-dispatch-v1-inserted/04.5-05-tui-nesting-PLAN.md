<objective>
Build a custom Ink component that renders sub-agent transcripts as a collapsible block under the parent's Agent tool call. Header line shows persona + truncated prompt; body shows each child turn (tool name + abbreviated result) indented + dimmed; default state is COLLAPSED. Decouple the component from pi's event shape via a thin `subagent-event-bridge` adapter.

**Wave 1, depends_on=[] (I1 fix):** This plan was originally placed in Wave 2 with `depends_on: [04.5-01, 04.5-03]` — overly conservative. The bridge consumes ONLY generic pi `AgentSessionEvent` shapes (`tool_call_start`, `tool_call_end`, `agent_end`) which are part of the pi-mono 0.68 SDK surface (NOT created by Plans 04.5-01 / 04.5-03). The Ink component consumes the bridge's pure-data ChildEventSnapshot type. Neither file imports from `@emmy/tools` `subagent/` modules nor from any OTel / governor / sidecar code. Promoting to Wave 1 means this plan can run in parallel with 04.5-01/02/03/04 — it shaves wall-clock off Wave 2.

Purpose: Pitfall #18 (sub-agent black-boxes) requires sub-agent steps be VISIBLE — not just observable in OTel traces but visible to the user in the terminal during a session. Without this UI, the trace tree is the only debug surface and the operator has to leave the terminal to inspect a sub-agent run. INTEGRATION-SKETCH §6 risk #2 calls this out as the only piece pi-mono doesn't help with — real engineering, not config.

Output: New Ink component + event bridge + tests. WIRING into the actual pi-emmy TUI session is a follow-up (depends on which extension hook pi 0.68 exposes for nested-turn rendering — likely the same `before_provider_request` extension factory used in pi-emmy-extension.ts). For v1 scope this plan delivers the component + bridge ready for wiring; the wiring task is captured as a `// TODO(04.5-07-or-later)` marker so Plan 04.5-07's E2E will exercise the UI manually.

       ```typescript
       export interface ChildTurnSnapshot {
         toolName: string;
         argsPreview: string;
         resultPreview: string;
         status: "running" | "ok" | "error";
       }
       export interface ChildEventSnapshot {
         persona: string;
         promptPreview: string;
         turns: ChildTurnSnapshot[];
         done: boolean;
         finalText?: string;
       }

       const PREVIEW_PROMPT_MAX = 60;
       const PREVIEW_ARGS_MAX = 80;
       const PREVIEW_RESULT_MAX = 200;

       function trunc(s: string, max: number): string {
         if (s.length <= max) return s;
         return s.slice(0, max - 1) + "…";
       }

       export function subscribeChildSession(
         session: { subscribe: (h: (e: any) => void) => () => void },
         persona: string,
         promptPreview: string,
         onUpdate: (snapshot: ChildEventSnapshot) => void,
       ): () => void {
         const turnsByCallId = new Map<string, ChildTurnSnapshot>();
         const orderedCallIds: string[] = [];
         let done = false;
         let finalText: string | undefined;

         const emit = () => {
           onUpdate({
             persona,
             promptPreview: trunc(promptPreview, PREVIEW_PROMPT_MAX),
             turns: orderedCallIds.map((id) => turnsByCallId.get(id)!),
             done,
             finalText,
           });
         };

         const unsub = session.subscribe((evt: any) => {
           if (!evt?.type) return;
           if (evt.type === "tool_call_start") {
             const id = evt.toolCallId ?? `_anon_${orderedCallIds.length}`;
             const turn: ChildTurnSnapshot = {
               toolName: String(evt.toolName ?? "unknown"),
               argsPreview: trunc(JSON.stringify(evt.args ?? {}), PREVIEW_ARGS_MAX),
               resultPreview: "",
               status: "running",
             };
             turnsByCallId.set(id, turn);
             orderedCallIds.push(id);
             emit();
           } else if (evt.type === "tool_call_end") {
             const id = evt.toolCallId ?? orderedCallIds[orderedCallIds.length - 1];
             const turn = turnsByCallId.get(id);
             if (turn) {
               const result = evt.result ?? {};
               const ok = !(result && typeof result === "object" && (result.ok === false || "error" in result));
               turn.status = ok ? "ok" : "error";
               const previewSrc =
                 (result && typeof result === "object" && "output" in result)
                   ? String((result as any).output ?? "")
                   : JSON.stringify(result ?? {});
               turn.resultPreview = trunc(previewSrc, PREVIEW_RESULT_MAX);
               emit();
             }
           } else if (evt.type === "agent_end") {
             done = true;
             const messages = evt.messages ?? [];
             for (let i = messages.length - 1; i >= 0; i--) {
               const m = messages[i];
               if (m?.role === "assistant" && Array.isArray(m.content)) {
                 finalText = m.content
                   .filter((c: any) => c?.type === "text")
                   .map((c: any) => String(c.text ?? ""))
                   .join("");
                 break;
               }
             }
             emit();
           }
         });
         return unsub;
       }
       ```

       ```tsx
       import React, { useState } from "react";
       import { Box, Text } from "ink";
       import type { ChildEventSnapshot, ChildTurnSnapshot } from "./subagent-event-bridge";

       export interface SubagentBlockProps {
         snapshot: ChildEventSnapshot;
         initialCollapsed?: boolean;
         onToggle?: () => void;
       }

       const STATUS_GLYPH: Record<ChildTurnSnapshot["status"], string> = {
         running: "…",
         ok: "✓",
         error: "✗",
       };

       const STATUS_COLOR: Record<ChildTurnSnapshot["status"], string> = {
         running: "cyan",
         ok: "green",
         error: "red",
       };

       export const SubagentBlock: React.FC<SubagentBlockProps> = ({
         snapshot,
         initialCollapsed = true,
         onToggle,
       }) => {
         const [collapsed, setCollapsed] = useState(initialCollapsed);
         const headerGlyph = snapshot.done ? "■" : collapsed ? "▶" : "▼";
         const handleToggle = () => {
           setCollapsed((c) => !c);
           onToggle?.();
         };
         // Header line: glyph Agent: <persona> — "<promptPreview>"
         const header = (
           <Text>
             <Text>{headerGlyph} </Text>
             <Text bold>Agent: {snapshot.persona}</Text>
             <Text dimColor> — "{snapshot.promptPreview}"</Text>
           </Text>
         );
         if (collapsed) {
           return <Box flexDirection="column">{header}</Box>;
         }
         return (
           <Box flexDirection="column">
             {header}
             {snapshot.turns.map((t, i) => (
               <Box key={i} marginLeft={2}>
                 <Text color={STATUS_COLOR[t.status]} dimColor>
                   {STATUS_GLYPH[t.status]} {t.toolName}({t.argsPreview})
                   {t.status !== "running" ? ` → ${t.resultPreview}` : ""}
                 </Text>
               </Box>
             ))}
             {snapshot.done && snapshot.finalText !== undefined ? (
               <Box marginLeft={2}>
                 <Text dimColor>final: "{snapshot.finalText.slice(0, 200)}"</Text>
               </Box>
             ) : null}
           </Box>
         );
       };
       ```
