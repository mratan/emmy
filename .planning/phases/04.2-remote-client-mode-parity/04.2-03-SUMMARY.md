## Decisions Made

- **eventsource-parser v3 API form** — RESEARCH.md and PATTERNS.md snippets used the v2 `createParser((event) => {...})` callback-as-positional-arg form; v3.0.x changed to `createParser({onEvent: (event) => {...}})`. Fixed inline (Rule 1 - bug). The `event.type !== "event"` guard was also dropped because v3's `onEvent` only fires for events (not comments/retries), making the guard unreachable.
- **Default base URL `http://127.0.0.1:8003`** — fallback when EMMY_SERVE_URL unset. Supports Spark-side smoke testing without env override, while still satisfying T-04.2-S5 because :8003 will fail-fast if no sidecar listening (returns `{exit: 1}`, never falls back to spawn).
- **Trailing slash stripped** — `baseUrl.replace(/\/$/, "")` keeps the joined URL canonical regardless of whether the operator supplies `https://spark.tailnet.ts.net:8003` or `https://spark.tailnet.ts.net:8003/`.
- **Lazy import** — `await import("./profile-swap-runner-http")` only resolves when EMMY_REMOTE_CLIENT='1'. Local mode pays zero import cost; Bun's module cache means subsequent remote calls are O(1) (T-04.2-Dispatch1 accepted).
