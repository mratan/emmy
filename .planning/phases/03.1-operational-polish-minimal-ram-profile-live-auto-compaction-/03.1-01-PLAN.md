1. **RAM headroom** (D-29): Drop vLLM `gpu_memory_utilization` 0.88 → 0.55 on a new v3.1 profile. Keep `max_model_len=131072` (user constraint: don't reduce model capability). Target emmy-serve footprint ~50-70 GB, leaving >40 GB system RAM free.

3. **Manual escape valves** (D-31, D-32): Register two pi slash commands. `/compact [args]` calls `ctx.compact({customInstructions})`; `/clear` calls (confirm + abort + waitForIdle + newSession) to reset the session from an over-full state without killing pi-emmy.
