Ship self-hosted SearxNG as emmy's one and only authorized local-first egress channel. This refines the project thesis from "NO OUTBOUND TRAFFIC" to "NO CLOUD INFERENCE + loopback-only research egress via a single auditable component". After this plan, the agent can research the open web in a daily-driver loop without breaking the air-gap discipline that makes emmy reproducible.

Deliverables per D-33..D-37:
1. `observability/searxng/` sibling of `observability/langfuse/` — digest-pinned Docker Compose stack with Google + DDG + Brave + Bing engines; loopback-only bind.
