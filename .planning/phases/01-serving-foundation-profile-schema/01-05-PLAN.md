**JSONL schema (one line per turn):**

```json
{"turn": 1, "role": "user", "content": "Please show me the first 20 lines of /tmp/foo.py.", "_expected_tool_call": {"name": "read", "args": {"path": "/tmp/foo.py", "end_line": 20}}}
{"turn": 2, "role": "assistant", "content": null, "tool_calls": [{"id": "call_read_1", "type": "function", "function": {"name": "read", "arguments": "{\"path\": \"/tmp/foo.py\", \"end_line\": 20}"}}]}
{"turn": 3, "role": "tool", "tool_call_id": "call_read_1", "content": "def foo():\n    return 42\n\ndef bar():\n    return 43\n"}
{"turn": 4, "role": "assistant", "content": "The file defines two functions: foo() returns 42 and bar() returns 43."}
```

Every subsequent group follows the same 4-turn shape. Context-growing turns (43-50) are eight user turns that reference back to turns 1-42 (e.g. "earlier you mentioned...", "in the foo.py we looked at..."), growing history length.

Rules the file MUST satisfy:
- Exactly 50 lines when `wc -l air_gap/session.jsonl` is run
- Every line parses via `json.loads`
- Every row has `turn` (int) and `role` (one of `user`/`assistant`/`tool`)
- Every `assistant` turn with `tool_calls` has exactly one entry with a valid `function.name` ∈ {read, write, edit, bash, grep, find, ls, web_fetch}
- Across all assistant turns, union of `function.name` values contains all 8 tool types
- At least 5 user turns in the 43-50 range reference earlier content (history accumulation)
- `web_fetch` tool-result turns contain the comment `"(mock — no network in air-gap)"` at the beginning of the `content` so it's unmistakable that no network call happened

```python
"""D-12 layered air-gap probes."""
from __future__ import annotations
import json, subprocess
from dataclasses import dataclass, field


@dataclass(frozen=True)
class LayerResult:
    layer: str                  # "a" | "b" | "c" | "d"
    name: str
    passed: bool
    detail: str                 # evidence / error message
    commands_run: list[str] = field(default_factory=list)


def _run(cmd: list[str], *, timeout: float = 10.0) -> tuple[int, str]:
    try:
        r = subprocess.run(cmd, capture_output=True, text=True, timeout=timeout)
        return r.returncode, (r.stdout or "") + (r.stderr or "")
    except subprocess.TimeoutExpired:
        return -1, f"timeout after {timeout}s"
    except FileNotFoundError:
        return -2, f"command not found: {cmd[0]}"


def layer_a_network_devices(container: str) -> LayerResult:
    """(a) Container has no non-loopback network devices."""
    cmds: list[str] = []
    cmd1 = ["docker", "inspect", container, "--format", "{{json .NetworkSettings.Networks}}"]
    cmds.append(" ".join(cmd1))
    rc1, out1 = _run(cmd1)
    if rc1 != 0:
        return LayerResult("a", "network_devices", False, f"docker inspect failed: {out1}", cmds)
    # Expect {"none":{...}} only
    try:
        nets = json.loads(out1.strip())
    except json.JSONDecodeError:
        return LayerResult("a", "network_devices", False, f"invalid json from docker inspect: {out1}", cmds)
    if list(nets.keys()) != ["none"]:
        return LayerResult("a", "network_devices", False, f"expected only 'none' network, got {list(nets.keys())}", cmds)

    cmd2 = ["docker", "exec", container, "ip", "-o", "addr"]
    cmds.append(" ".join(cmd2))
    rc2, out2 = _run(cmd2)
    if rc2 != 0:
        return LayerResult("a", "network_devices", False, f"ip addr failed: {out2}", cmds)
    # Accept lines that reference "lo" interface; reject any eth*, ens*, wlan*
    forbidden = ("eth", "ens", "wlan", "enp", "wlp")
    for line in out2.splitlines():
        parts = line.split()
        if len(parts) >= 2 and any(parts[1].startswith(f) for f in forbidden):
            return LayerResult("a", "network_devices", False, f"forbidden interface present: {line}", cmds)
    return LayerResult("a", "network_devices", True, "only loopback present", cmds)


def layer_b_dns_audit(container: str) -> LayerResult:
    cmds = []
    cmd1 = ["docker", "exec", container, "cat", "/etc/resolv.conf"]
    cmds.append(" ".join(cmd1))
    rc1, out1 = _run(cmd1)
    # If resolv.conf has any external nameserver (not 127.x.x.x), fail
    allowed_prefixes = ("127.", "#", "")
    bad = []
    for line in (out1 or "").splitlines():
        s = line.strip()
        if s.startswith("nameserver"):
            ns = s.split()[-1]
            if not any(ns.startswith(p) for p in allowed_prefixes):
                bad.append(ns)
    if bad:
        return LayerResult("b", "dns_audit", False, f"external nameservers present: {bad}", cmds)

    cmd2 = ["docker", "exec", container, "getent", "hosts", "huggingface.co"]
    cmds.append(" ".join(cmd2))
    rc2, out2 = _run(cmd2, timeout=5.0)
    # Must FAIL (non-zero return) — DNS should be unreachable in air-gap
    if rc2 == 0:
        return LayerResult("b", "dns_audit", False,
                           f"getent resolved huggingface.co (should fail in air-gap): {out2.strip()}", cmds)
    return LayerResult("b", "dns_audit", True, "no external DNS reachable", cmds)


def layer_c_telemetry_env(container: str) -> LayerResult:
    cmds = []
    missing = []
    for var, expected in (("VLLM_NO_USAGE_STATS", "1"), ("DO_NOT_TRACK", "1")):
        cmd = ["docker", "exec", container, "printenv", var]
        cmds.append(" ".join(cmd))
        rc, out = _run(cmd)
        if rc != 0 or out.strip() != expected:
            missing.append(f"{var}={out.strip() or '<unset>'} (expected {expected})")
    if missing:
        return LayerResult("c", "telemetry_env", False, "; ".join(missing), cmds)
    return LayerResult("c", "telemetry_env", True, "VLLM_NO_USAGE_STATS=1 DO_NOT_TRACK=1", cmds)


def layer_d_hf_offline_env(container: str) -> LayerResult:
    cmds = []
    missing = []
    for var, expected in (("HF_HUB_OFFLINE", "1"), ("TRANSFORMERS_OFFLINE", "1")):
        cmd = ["docker", "exec", container, "printenv", var]
        cmds.append(" ".join(cmd))
        rc, out = _run(cmd)
        if rc != 0 or out.strip() != expected:
            missing.append(f"{var}={out.strip() or '<unset>'} (expected {expected})")
    if missing:
        return LayerResult("d", "hf_offline_env", False, "; ".join(missing), cmds)
    return LayerResult("d", "hf_offline_env", True, "HF_HUB_OFFLINE=1 TRANSFORMERS_OFFLINE=1", cmds)
```

**2. `emmy_serve/airgap/validator.py`** — run all 4 layers + emit JSON report:

```python
"""Airgap validator CLI: run all 4 D-12 layers against running container; emit JSON report + exit code."""
from __future__ import annotations
import argparse, json, sys, time
from dataclasses import dataclass, field, asdict
from pathlib import Path
from .probe import (
    LayerResult, layer_a_network_devices, layer_b_dns_audit,
    layer_c_telemetry_env, layer_d_hf_offline_env,
)
from ..diagnostics.atomic import write_json_atomic


@dataclass(frozen=True)
class AirGapReport:
    ts: str
    container: str
    layers: list[LayerResult]
    passes: bool
    failures: list[str]


def run_airgap_probe(container: str = "emmy-serve") -> AirGapReport:
    results = [
        layer_a_network_devices(container),
        layer_b_dns_audit(container),
        layer_c_telemetry_env(container),
        layer_d_hf_offline_env(container),
    ]
    failures = [f"layer ({r.layer}) {r.name} FAILED: {r.detail}" for r in results if not r.passed]
    return AirGapReport(
        ts=time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        container=container,
        layers=results,
        passes=len(failures) == 0,
        failures=failures,
    )


def _cmd_pre_boot(args) -> int:
    """Pre-boot: verify serving.yaml's env section alone (does not require a running container)."""
    from ..profile.loader import load_serving
    from ..profile.schema import ProfileConfigError
    try:
        serving = load_serving(Path(args.profile) / "serving.yaml")
    except ProfileConfigError as e:
        print(f"pre-boot FAIL: {e}", file=sys.stderr)
        return 1
    env = serving.env
    probs = []
    if env.VLLM_NO_USAGE_STATS != "1":
        probs.append("env.VLLM_NO_USAGE_STATS != '1'")
    if env.DO_NOT_TRACK != "1":
        probs.append("env.DO_NOT_TRACK != '1'")
    if env.HF_HUB_OFFLINE != "1":
        probs.append("env.HF_HUB_OFFLINE != '1'")
    if env.TRANSFORMERS_OFFLINE != "1":
        probs.append("env.TRANSFORMERS_OFFLINE != '1'")
    if probs:
        for p in probs:
            print(f"pre-boot FAIL: {p}", file=sys.stderr)
        return 1
    print("pre-boot OK: serving.yaml env section passes D-12 (c)/(d) policy")
    return 0


def _dump_report(report: AirGapReport, out_path: Path | None) -> None:
    serializable = {
        "ts": report.ts,
        "container": report.container,
        "passes": report.passes,
        "failures": report.failures,
        "layers": [asdict(r) for r in report.layers],
    }
    if out_path:
        write_json_atomic(out_path, serializable)
    print(json.dumps(serializable, indent=2))


def _cmd_post_boot(args) -> int:
    report = run_airgap_probe(container=args.container)
    _dump_report(report, Path(args.out) if args.out else None)
    if not report.passes:
        for f in report.failures:
            print(f, file=sys.stderr)
        return 1
    return 0


def main(argv: list[str] | None = None) -> int:
    p = argparse.ArgumentParser(prog="airgap_probe")
    sub = p.add_subparsers(dest="cmd", required=True)

    pre = sub.add_parser("pre-boot")
    pre.add_argument("--profile", required=True)
    pre.set_defaults(_handler=_cmd_pre_boot)

    post = sub.add_parser("post-boot")
    post.add_argument("--container", default="emmy-serve")
    post.add_argument("--out", default=None, help="write JSON report here")
    post.set_defaults(_handler=_cmd_post_boot)

    args = p.parse_args(argv)
    return args._handler(args)


if __name__ == "__main__":
    sys.exit(main())

#!/usr/bin/env bash
# .githooks/pre-commit — Layer 2 of PROFILE-06 enforcement (RESEARCH.md §5).
#
# Activate via: git config core.hooksPath .githooks
#
# Validates profile bundles touched by staged changes. Fails the commit if any
# bundle's computed hash differs from its stored profile.yaml.hash.

set -euo pipefail

# Find profile bundles that have staged changes.
changed=$(git diff --cached --name-only | grep '^profiles/[^/]*/v[0-9]*/' || true)
if [[ -z "$changed" ]]; then
  exit 0
fi

bundles=$(echo "$changed" | awk -F/ '{print $1"/"$2"/"$3}' | sort -u)

for bundle in $bundles; do
  echo "[pre-commit] validating $bundle"
  if ! uv run emmy profile validate "$bundle"; then
    echo "[pre-commit] FAIL: $bundle failed validation (schema/hash/policy)." >&2
    echo "[pre-commit] Bypass with --no-verify at your own risk." >&2
    echo "[pre-commit] Recompute with: uv run emmy profile hash $bundle --write" >&2
    exit 1
  fi
done
exit 0

```markdown
# Profile Immutability (PROFILE-06) — 3-Layer Enforcement

Profiles under `profiles/<name>/v<N>/` are immutable. Any field change creates a new
version directory (`v<N+1>/`). Three layers enforce this:

## Layer 1 — Validator (always on)

`emmy profile validate <path>` recomputes the bundle hash via the canonicalization
algorithm in `emmy_serve.profile.hasher` and compares to `profile.yaml.hash`. Exits:

- 0 — schema + hash OK
- 1 — schema error (missing field, wrong type, unknown key)
- 2 — hash mismatch (D-03 violation)
- 3 — canonicalization error (symlink, non-UTF-8, disallowed dotfile)
- 4 — cross-field policy failure (e.g. `env.VLLM_NO_USAGE_STATS != "1"`)

This layer is invoked every time `scripts/start_emmy.sh` runs as a pre-flight check
and by every CI/hook layer below.

## Layer 2 — Git pre-commit hook (recommended for developers)

`.githooks/pre-commit` invokes `emmy profile validate` on every profile bundle
touched by staged changes. Activate once:

```bash
git config core.hooksPath .githooks
```

Bypass with `git commit --no-verify` if you're deliberately committing a
recompute-and-hash change (the hook will still pass once `profile.yaml.hash` is
updated). Hook prints the exact command to recompute.

## Layer 3 — CI enforcement (self-hosted + cloud)

`.github/workflows/airgap.yml` has a `profile-hash-integrity` job that runs
`emmy profile validate` on every `profiles/*/v*/` on every PR touching
`profiles/**` or `emmy_serve/**`. The job runs on a GitHub-hosted runner (no
GPU required). Any mismatch fails the PR.

This layer is what makes `--no-verify` harmless at the project level — a
developer can bypass locally but cannot merge without CI blessing.

## Why not `chattr +i`?

Filesystem immutable bit requires root, breaks `git gc`, doesn't protect
against `rm -rf + recreate`, and is OS-specific. The 3-layer validator chain
is strictly better for a single-user tool + a research artifact reviewed via git.

## Recomputing after legitimate edits

The finder (`scripts/find_kv_budget.py`) and thermal replay's
`--record-floors` mode are the two Phase-1 code paths that legitimately mutate
v1/ after its initial commit. Both recompute the hash (`emmy profile hash --write`)
as part of their normal flow. PRs that land a profile mutation MUST cite the
run-dir artifact that produced the mutation in `PROFILE_NOTES.md`; CI catches
drift through the hash integrity job.
```
