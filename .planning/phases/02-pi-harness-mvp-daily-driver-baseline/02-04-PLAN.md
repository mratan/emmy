**loadProfile(profileDir):**
- Reads `${profileDir}/profile.yaml`, `serving.yaml`, `harness.yaml` via js-yaml.
- Obtains authoritative hash from `profile.yaml.hash` OR by shelling `uv run emmy profile hash <profileDir>` (CONTEXT.md §code_context — do not reimplement the hasher).
- Returns a `ProfileSnapshot` (from `@emmy/provider`) populated from the parsed YAMLs.
- Non-existent profileDir → throws ProfileLoadError with "not found".
- YAML parse error → ProfileLoadError with dotted-path.
- **W4 FIX: Missing `serving.engine.max_model_len`** → throws `ProfileLoadError("<path>/serving.yaml", "engine.max_model_len: required field missing")`. Tested explicitly in a RED case.
- **B3 FIX: Parses `harness.tools.grammar` as nested `{ path: string, mode: "reactive" | "disabled" } | null`.** If the field is absent or null in YAML, loader returns `null`. If present, `path` MUST be a string AND `mode` MUST be `"reactive"` or `"disabled"` — any other shape throws `ProfileLoadError` with dotted path. **The loader does NOT accept a plain-string `tools.grammar` — that is the pre-revision flattened shape and is rejected.**
- The loader is forgiving on optional fields (top_p, stop, strip_fields) but strict on required ones (served_model_name, max_model_len, temperature, top_p as defaults, max_tokens).

**assemblePrompt({profileSystemMd, agentsMd, agentsMdPath, toolDefsText, userPrompt?}):**
- Returns `{text, sha256, layers, agents_md_path}`.
- Layer order FIXED: system.md, AGENTS.md, tool_defs, user (CONTEXT-04).
- `agentsMd === null` → AGENTS.md layer emitted as `{present: false, tokens_approx: 0}`.
- `text` = `\n\n`-joined concatenation.
- `sha256` = full 64-char hex SHA-256 of `text`.
- `tokens_approx` per layer = `Math.ceil(layer_text.length / 4)`.
- Emits `emitEvent({event: "prompt.assembled", ts, sha256, layers})`.
- Writes `prompt.assembled sha256=<hex>` to stderr on every call (HARNESS-06).

**runSpOk(baseUrl, servedModelName):**
- Imports `postChat` from `"@emmy/provider"` (bare package name — W1 re-export).
- Sends payload matching `emmy_serve/canary/sp_ok.py` byte-for-byte; `chat_template_kwargs` at TOP LEVEL.
- Returns `{ok, responseText}`; does NOT throw on `ok === false`. Caller throws SpOkCanaryError.
- Timeout 60_000ms.
- Tests read `/data/projects/emmy/emmy_serve/canary/sp_ok.py` via readFileSync + regex extract of the constant and assert byte equality with the TS constant.

**computeMaxInputTokens({measured_gpu_memory_utilization, max_model_len, output_reserve_tokens}):**
- Formula: `max_input_tokens = max_model_len - output_reserve_tokens`, non-negative, rounded down.
- Derivation string includes both numbers.
- `measured_gpu_memory_utilization < 0.5` → MaxModelLenError.
- `max_model_len <= 0` → MaxModelLenError.
- `output_reserve_tokens >= max_model_len` → MaxModelLenError.

function parseGrammarConfig(raw: unknown, at: string): { path: string; mode: "reactive" | "disabled" } | null {
  if (raw === null || raw === undefined) return null;
  if (typeof raw === "string") {
    throw new ProfileLoadError(at, `must be a mapping with path + mode, or null (pre-revision flattened-string shape is rejected; see CONTEXT D-11)`);
  }
  if (typeof raw !== "object" || Array.isArray(raw)) {
    throw new ProfileLoadError(at, `must be a mapping with keys {path, mode}, or null`);
  }
  const obj = raw as Record<string, unknown>;
  if (typeof obj.path !== "string" || !obj.path.trim()) {
    throw new ProfileLoadError(`${at}.path`, `must be a non-empty string (relative to profile dir)`);
  }
  if (obj.mode !== "reactive" && obj.mode !== "disabled") {
    throw new ProfileLoadError(`${at}.mode`, `must be "reactive" or "disabled" (got ${JSON.stringify(obj.mode)})`);
  }
  return { path: obj.path, mode: obj.mode };
}

function loadYaml(path: string): any {
  if (!existsSync(path)) throw new ProfileLoadError(path, "file not found");
  try { return yaml.load(readFileSync(path, "utf8")); }
  catch (e) { throw new ProfileLoadError(path, `YAML parse: ${e instanceof Error ? e.message : String(e)}`); }
}

function requireStr(obj: any, at: string, keys: string[]): string {
  const v = keys.reduce((o, k) => (o && typeof o === "object" ? o[k] : undefined), obj);
  if (typeof v !== "string" || !v.trim()) throw new ProfileLoadError(at, "must be a non-empty string");
  return v;
}
function requireNum(obj: any, at: string, keys: string[]): number {
  const v = keys.reduce((o, k) => (o && typeof o === "object" ? o[k] : undefined), obj);
  if (typeof v !== "number" || !Number.isFinite(v)) throw new ProfileLoadError(at, "must be a number");
  return v;
}
function samplingTopK(servingYaml: any): number | undefined {
  const v = servingYaml?.sampling_defaults?.top_k;
